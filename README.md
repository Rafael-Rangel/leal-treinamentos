# Leal Treinamentos — Site Institucional

Landing page multi-página da **Leal Treinamentos** (segurança do trabalho, Rio de Janeiro).

## Páginas

- `index.html` — Home
- `cursos.html` — Catálogo de cursos
- `corporativo.html` — Soluções B2B
- `sobre.html` — Sobre nós
- `contato.html` — Contato
- `privacidade.html` — Política de Privacidade (LGPD)
- `termos.html` — Termos de Uso

## Estrutura

```
leal-treinamentos/
├── index.html
├── css/main.css
├── js/
│   ├── main.js
│   ├── animations.js
│   ├── layout.js
│   ├── cookies.js
│   └── images.js
└── assets/clientes/
```

## Executar localmente

```bash
cd leal-treinamentos
python -m http.server 8080
```

Acesse: http://localhost:8080
